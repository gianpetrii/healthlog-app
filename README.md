# HealthLog App

## Descripción del proyecto

Plataforma para **historial médico familiar**: subir y organizar documentos, perfiles por integrante y flujos de información de salud, con Next.js y Firebase.

## Problema que resuelve

Atiende el caos de estudios, recetas e indicaciones repartidas entre carpetas, WhatsApp y distintos centros: ofrece un repositorio compartido y controlado para familias que coordinan el cuidado de varios miembros.

## Stack

- Next.js, React, TypeScript  
- Firebase  

## Requisitos

- Node.js LTS  

## Instalación

```bash
npm install
npm run dev
```

Scripts: `build`, `start`, `lint`.

## Variables de entorno

Configurá `.env.local` con las variables `NEXT_PUBLIC_FIREBASE_*` (y otras que indique el código) para tu proyecto Firebase.
